A programming language is a vocabulary and collection of grammatical rules for guidance to execute particular tasks on a machine or computing system. The term programming language usually refers to high-level languages, such as BASIC, C, C++, COBOL, Java, FORTRAN, Ada, and Pascal. Each programming language has a unique set of keywords (words that it understands) and a special syntax for organizing program instructions.
